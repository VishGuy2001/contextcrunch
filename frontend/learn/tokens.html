How each model tokenizes and handles context

Claude
Full recall up to the context limit — re-reads everything every message. Tells you when it hits the limit. Opus 4.7 uses a new tokenizer producing up to 35% more tokens than older Claude models for the same text.
Haiku 4.5: 200k · Sonnet 4.6: 1M · Opus 4.7: 1M
Custom BPE · ~3.5 chars/token (Haiku/Sonnet) · ~2.6 (Opus 4.7)
ChatGPT
Silent truncation — drops oldest messages without telling you when the window fills. The only model with exact public token counts via tiktoken (cl100k).
Free (GPT-4o mini): 32k · Plus (GPT-4o): 272k · Pro: 1.05M
cl100k BPE · ~4.0 chars/token · vocab 100,277
Gemini
Largest vocabulary = fewest tokens per word, so it uses its window more efficiently. Rate limits hit before token limits on free tier — you'll see quota errors before context errors.
Free (2.5 Flash): 1M · Pro (2.5 Pro): 1M
SentencePiece unigram · ~4.5 chars/token · vocab 256,000
